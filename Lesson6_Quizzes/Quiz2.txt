1.  Name two methods you can use to add a node to the DOM tree, and explain the difference between them.
You can use the appendChild() method or the insertBefore() method.
The appendChild method is used by specifying the node to be attached, which is the child node, and the parent node to attach it to. 
The insertBefore method allows you to specify the order of the nodes.
